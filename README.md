## About
FastAPI backend for LifeSync app with ML recommendation and analysis support

## Features
- Analyze academics, mental health, vitals, nutrition, activities, workouts, sleep etc...
- Analyze data for each day, month, year etc... and find trends
- Get ML recommendations and trends

## Tech stack used
- Python
- FastAPI
- PyDantic
- Typing

## Running
Run `fastapi dev`
